# 3 증권사 프로그램 설치 및 소개(Open API, KOA Studio)
- openapi? 
  - 주식 데이터 수집
  - 증권사 계좌 연동 
  - 원하는 시점에 거래
- ex : 키움
  - 공인인증서 발금
    - 씨발 개같이 찾기 힘듦
    - 이건 증권용 이거나 키움전용 인증서같음...   
    암튼 새로 신청 필요
    - 무료용 인증서로 신청 : 용도 제한용
  - openAPI 
    - 사용신청
      - 걍 등록 누르면됨 
    - 다운로드 (키움 Open API+ 모듈 다운로드)후 설치
  - KOA Studio
    - 실제 증권사 데이터를 확인하는 프로그램
    - 받으면 에러남
      - mfc100.dll 없다함
      - koa 압축 풀떄 있던 파일 다 아까 설치한 openAPI 폴더에 넣고 다시 실행하면 됨
    - 로그인은 모의투자 체크한 상태로 ID/PW만 ㄱㄱ
      - 패치 신경써서 할것.
    - 로그인 정상적으로 되면 작업표시줄 아이콘에 생김
    - 계좌 비밀번호 저장 (이건 임시계좌에 임시 비번인듯함.)
